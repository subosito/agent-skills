# agent-skills

Personal collection of [agent skills](https://agentskills.io).

## slidev + neversink

[Slidev](https://sli.dev) is presentation slides for developers and [Neversink](https://gureckis.github.io/slidev-theme-neversink/) is one of its best theme.

```sh
$ npx skills add slidevjs/slidev
$ npx skills add subosito/agent-skills --skill slidev-neversink
```

## Ory

[Ory](https://www.ory.com) is a modern and modular IAM solution.

- Kratos: a headless cloud-native authentication and identity management.
- Hydra: a hardened, OpenID Certified OAuth 2.0 Server and OpenID Connect Provider.
- Keto: a scalable, performant authorization server for managing permissions at scale.

```
$ npx skills add subosito/agent-skills --skill ory-kratos
$ npx skills add subosito/agent-skills --skill ory-hydra
$ npx skills add subosito/agent-skills --skill ory-keto
```
